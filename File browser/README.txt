Κάθε εγγραφή πρέπει να περιέχει τα εξής:
  
- Όνομα της βιβλιοθήκης
  
- Κλάσεις που χρησιμοποιήθηκαν από τη βιβλιοθήκη
  
- path του .jar αρχείου μέσα στον κατάλογο αρχείων που υποβάλλετε










  

Χρησιμοποιω την βιβλιοθηκη WrapLayout. Ειναι μια κλαση η WrapLayout.java και την χρησιμοποιω
στην insideFrame.java και την searchFrame.java. Το path της στον καταλογο που υπεβαλα ειναι:

Πως μεταγλωττίζεται και εκτελείται ο κώδικας σας. Η μεταγλώττιση και η εκτέλεση πρέπει 
να γίνεται 
μέσω γραμμής της εντολών (command line). 
Οι επιλογές που έχετε είναι οι εξής:
  
- ant script (προτιμητέο)
  
- maven script
  
- gradle script
  
- δικό σας script

  
Σε όλες τις παραπάνω περιπτώσεις παραθέστε αναλυτικά τις εντολές μεταγλώττισης και εκτέλεσης του κώδικα σας.


Χρησιμοποιησα ant script. Για να το τρεξω πρεπει να παω με το command line στον φακελο που υπεβαλα και
να πατησω την εντολη ant.
